# Spark DataFrame Practice

This folder contains my practice work for Apache Spark using PySpark DataFrames.

The main goal of this assignment is to understand how Spark DataFrames are created, inspected, cleaned, transformed, and joined. I focused on writing clear code and checking the data carefully before applying transformations, because incorrect schema or bad data can affect the final results.

Final Notes

This assignment helped me practice the basic and important parts of working with Spark DataFrames. I tried to keep the notebooks simple, clear, and focused on the required tasks.

The main focus was not only to write Spark code, but also to understand why each step is needed, especially when dealing with schema, null values, duplicate records, and join results.
