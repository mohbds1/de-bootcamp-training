# 🚀 Week 1: Python Fundamentals, OOP, and Modules

Welcome to the **Week 1** repository of the Data Engineering Bootcamp. This week focuses on solidifying advanced Python concepts, project structuring, and Object-Oriented Programming (OOP) principles through practical tasks and a mini-project.

---

## 📂 Tasks and Projects Overview

### 1. 🧬 Task 1: Multiple Inheritance (`Task_1.ipynb`)
A hands-on Jupyter Notebook exploring the concept of multiple inheritance in Python.
- **Objective:** Understand Method Resolution Order (MRO) and how Python handles classes inheriting from multiple parent classes.
- **Concepts:** OOP, Class Hierarchies, Inheritance.

### 2. 📦 Task 2: Packages and Modules (`Task_2`)
A practical demonstration of organizing Python code into reusable modules and packages.
- **Objective:** Learn how to create custom modules, structure packages with `__init__.py`, and import them efficiently into a main script.
- **Concepts:** Project Structuring, Imports, Code Reusability.

### 3. 🎓 Project: Student Performance Analyzer (`Student-Performance-Analyzer`)
A fully functional Command-Line Interface (CLI) application built from scratch to analyze student records.
- **Objective:** Apply OOP principles to build a real-world application that reads, processes, and analyzes data from a CSV file.
- **Highlighted Skills:** Object-Oriented Programming (Classes & Objects), File I/O (CSV handling), Modular Architecture (`main.py`, `models.py`, `analytics.py`, `utils.py`).

---

## 🛠️ Core Technologies Used
- **Language:** Python 3.8+
- **Data Storage:** CSV Files
- **Environment:** Command Line (CLI) & Jupyter Notebooks
