# 🚀 Week 2: Data Wrangling, EDA, and NLP Fundamentals

Welcome to the **Week 2** repository of the Data Engineering Bootcamp. This week showcases a transition from raw, messy datasets to actionable insights, concluding with an introduction to Natural Language Processing (NLP) concepts. 

This directory serves as a mini-portfolio of hands-on projects focusing on **Data Cleaning, Exploratory Data Analysis (EDA), and Text Vectorization.**

---

## 📂 Projects Showcase

### 1. 👟 Nike Sales Data Cleaning (`nike-sales-data-cleaning`)
A comprehensive data wrangling project that tackles real-world, dirty sales data.
- **Objective:** Transform a highly unstructured Nike sales dataset into a clean, analysis-ready format.
- **Highlighted Skills:** Handling missing values, date parsing, string manipulation, data standardization, and automated audit reporting.
- **Tech Stack:** Python, Pandas, Jupyter Notebook.

### 2. ☕ Cafe Sales Data Analysis (`cafe-sales-data-analysis`)
An end-to-end Exploratory Data Analysis (EDA) project extracting business intelligence from daily cafe transactions.
- **Objective:** Cleanse transactional data and perform statistical analysis to uncover sales trends, peak operational hours, and top-performing products.
- **Highlighted Skills:** Data transformation, exploratory analysis, trend identification, and data visualization.
- **Tech Stack:** Python, Pandas, Jupyter Notebook.

### 3. 📰 Article Similarity Calculator (`article-similarity-calculator`)
A mini search engine and recommendation system .
- **Objective:** Build a system capable of parsing multiple articles, tokenizing text, and calculating the Cosine Similarity to find related content.
- **Highlighted Skills:** Text preprocessing (Tokenization, Stopwords removal), TF-IDF Vectorization, and building baseline recommendation models.
- **Tech Stack:** Python, Pandas, Numpy.

### 4. 🧮 Manual Text Vectorization (`document_vectorization.py`)
A standalone script demonstrating the mathematical foundation of NLP.
- **Objective:** A from-scratch implementation of the "Bag of Words" algorithm to understand how textual data is converted into numerical vectors before relying on high-level libraries.
