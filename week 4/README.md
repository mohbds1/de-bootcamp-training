# 🚀 Week 4: Data Extraction & Web Scraping

Welcome to the **Week 4** repository. This week focuses on data acquisition strategies, moving from basic scraping scripts to complex, multi-module data extraction and processing pipelines.

---

## 📂 Projects Portfolio

### 1. 🕷️ Multi-Level Scraper & Processor (`Books-Scraping`)
A high-level, modular scraping system designed to extract, process, and organize bookstore data automatically.
- **Objective:** Build a robust pipeline that scrapes book metadata, categorizes it by star rating, and organizes both text data and images into a structured directory system.
- **Key Architectural Modules:**
  - `scraper.py`: Handles HTTP requests and raw HTML parsing.
  - `processor.py`: Cleans and transforms extracted data into structured formats.
  - `organizer.py`: Automates directory management and categorizes records by rating.
- **Highlighted Skills:** Modular Python programming, automated file system organization, image processing, and complex data categorization.
- **Tech Stack:** Python, BeautifulSoup4, Requests, Pandas.

### 2. 📖 Basic Book Store Scraper (`BeautifulSoup-Task`)
An introductory project focused on the fundamentals of web scraping and data persistence.
- **Objective:** Navigate a bookstore website to extract essential metadata (titles, prices, ratings) and save them to a flat CSV file.
- **Highlighted Skills:** DOM traversal, CSS selector targeting, and basic media downloading.
- **Tech Stack:** Python, BeautifulSoup4, Requests, Jupyter Notebook.
